# Validation Rules

Replace with the real rules `scripts/validate.py` enforces.

Document each rule as:

- **Rule name** — what it checks.
- **Why** — the reason this rule exists.
- **On failure** — what message the script emits and how the skill body
  should surface it to the user.

Keep this file authoritative — the script and the body should both refer to
it instead of duplicating logic.
