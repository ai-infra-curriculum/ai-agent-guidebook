# Reference Notes

Replace this file with the actual reference material your skill needs.

Reference files are loaded by the model on demand — when a step in the
procedure says "consult references/X.md", the model reads that file at that
moment. This keeps the main SKILL.md body small while making detailed
material available when needed.

Good use cases for a reference file:

- Long lookup tables (e.g., command-per-package-manager).
- Schema definitions for structured outputs.
- Domain-specific conventions and terminology.
- Detailed failure-mode catalogs.
- Reusable rubrics for evaluation steps.
