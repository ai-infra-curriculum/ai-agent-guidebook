## Getting Help

### Before Asking for Help

1. **Check the documentation**
   - Read relevant guides in `guides/`
   - Check templates for examples
   - Review comparison matrix for tool selection

2. **Search existing issues**
   - [GitHub Issues](https://github.com/ai-infra-curriculum/ai-agent-guidebook/issues)
   - Someone may have already asked your question

3. **Check official documentation**
   - [Claude Code Docs](https://docs.claude.com/claude-code)
   - [GitHub Copilot Docs](https://docs.github.com/copilot)
   - [Gemini API Docs](https://ai.google.dev/docs)

### What to Include When Asking for Help

**For documentation questions:**
- Which guide you're reading
- What you're trying to accomplish
- What's unclear or confusing
- What you've already tried

**For technical issues:**
- What you're trying to do
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your environment (OS, tool versions, etc.)
- Error messages (full text)
- Relevant configuration files

## Common Issues

### Documentation Issues

**Issue**: Can't find information about specific topic

**Solutions**:
- Check table of contents in README
- Use GitHub search: Press `/` and type your query
- Check related guides (may be in different section)
- Open an issue requesting the documentation

**Issue**: Example code doesn't work

**Solutions**:
- Verify you have correct dependencies installed
- Check for version-specific differences
- Review error messages carefully
- Open an issue with details about the problem

**Issue**: Broken links or outdated information

**Solutions**:
- Open an issue reporting the problem
- Submit a pull request with the fix
- Check official documentation for current info

### Tool-Specific Issues

**Claude Code not working?**
1. Check [Claude Code Troubleshooting](guides/claude-code/README.md#troubleshooting)
2. Verify MCP server configuration
3. Check official Claude Code docs
4. Open issue if documentation needs improvement

**GitHub Copilot not working?**
1. Check [Copilot Troubleshooting](guides/github-copilot/README.md#troubleshooting)
2. Verify subscription status
3. Check IDE extension installation
4. Review GitHub Copilot docs

**MCP Servers not working?**
1. Check [MCP Server Catalog](guides/mcp-servers/catalog.md)
2. Verify installation and configuration
3. Check server-specific documentation
4. Test with simple example
